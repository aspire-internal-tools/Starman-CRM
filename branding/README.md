# Branding

Aspire brand reference material and source brand assets for Starman CRM.

## Contents

| Folder | Purpose |
|---|---|
| `aspire-branding-skill/` | Source Aspire branding kit, including logo files, colour rules, typography guidance, and usage instructions. |

## Current Usage

The active Starman UI uses copied logo assets in:

- `design/assets/`
- `starman-app/web/assets/`

The brand rules are summarized for this CRM in `docs/Aspire-Brand-Standard.md`.

## Rules

- Treat this folder as source/reference material.
- Do not recolour, stretch, crop, or rebuild Aspire logos.
- Do not use gold as a large fill or body-text colour.
- Keep released Starman tools marked with product name, version, and date.
