# Sample Data

Example payloads used by the connector prototype and demo smoke test.

## Files

| File | Purpose |
|---|---|
| `meta_lead.json` | Example Meta lead payload. |
| `advisorstream_activity.csv` | Example AdvisorStream engagement export. |

## Rules

- Keep examples fictional.
- Do not store real lead, client, policy, account, or contact data.
- If adding samples, name them by source and purpose, for example `source_event-type.json`.
