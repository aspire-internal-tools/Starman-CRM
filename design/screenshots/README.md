# Screenshots

Visual evidence and design references captured during Starman UI iterations.

## Use

- Store screenshots that document visual state, regressions, or design decisions.
- Prefer descriptive names when adding new files:

```text
YYYY-MM-DD_screen-or-feature_state.png
```

Example:

```text
2026-07-14_client-360_documents-files.png
```
